Instant Ordering
-user select station
-user select product
-user create order
-station accept user's order
-rider pickup
-rider deliver refilled water
-station finish order

Cancel Ordering
-user/station cancel order in the pending order

Cancel Subscription
-user cancel subscription in the pending subscription

Subscribing
-user select station
-user create subscription
-station accept user's subscription
-rider pickup
-rider create order from subscription (with status already in confirmed, picked up, and status alredy in refilling status)
-rider deliver refilled water
-station finish order
-process would repeat for the remainder of subscription
